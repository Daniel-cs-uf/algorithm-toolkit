Hello This is a quick guide to my Gale-Shapley Algorithm program for assignment 1

One important thing to note is that for my program I use the non-standard parse library as it was allowed by the instructor in
a discussion post

To install the parse library in the terminal type python -m pip install parse
To confirm its installed in the terminal type python -c "import parse; print(parse.__version__)"

To run I used the Run/Debug feature in vs code, you will need to select a python interpreter to do this
Another method if this gives you trouble is in the integrated terminal type python galeShapley.py

this program takes in input from the test.txt file and the output should be put in the output.txt file
